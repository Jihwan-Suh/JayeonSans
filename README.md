# 자연산스 · Jayeon Sans

> 맑고 단정하면서도, 지나치게 반듯하지 않은 자연스러운 한글 산세리프

![자연산스 소개 페이지 미리보기](./preview.png)

## 소개

**자연산스**는 프리텐다드를 바탕으로 글자 형태를 다시 다듬어 만든 한글 산세리프입니다.

펴진고딕이 직선에서 오는 정돈되고 깔끔한 인상을 강조했다면, 자연산스는 자음 **ㅅ·ㅈ·ㅊ**의 형태를 새롭게 조정해 획의 흐름이 보다 자연스럽게 이어지도록 만들었습니다. 단정한 골격은 유지하면서도 지나치게 딱딱하지 않은, 편안하고 부드러운 인상을 지향합니다.

개인 작업으로 제작한 글꼴이라 현재 제공되는 **Regular**와 **Medium** 외에 추가 웨이트를 제작할 계획은 없습니다. 완성한 뒤 한동안 묵혀 두며 공개 여부를 고민했지만, 필요한 분들이 편하게 활용할 수 있기를 바라며 배포합니다.

---

## 특징

### 자연스러운 자음 형태

`ㅅ`, `ㅈ`, `ㅊ`의 형태를 다듬어 직선적인 산세리프에서 느껴질 수 있는 경직된 인상을 줄였습니다.

### 읽기 편한 균형감

작은 크기의 본문에서도 글자의 형태가 또렷하게 구분되도록 안정적인 비율과 리듬을 유지했습니다.

### 폭넓은 활용성

브랜딩, 문서, 편집 디자인, 웹사이트, 앱과 시스템 UI 등 다양한 환경에서 활용할 수 있습니다.

## 제공 웨이트

| 웨이트 | 굵기 | 파일 |
|---|---:|---|
| Regular | 400 | [`JayeonSans-Regular.ttf`](./JayeonSans-Regular.ttf) |
| Medium | 500 | [`JayeonSans-Medium.ttf`](./JayeonSans-Medium.ttf) |

> 개인 작업 여건상 추가 웨이트 제작은 예정되어 있지 않습니다.

## 글꼴 표본

### Regular · 400

> 산을 닮은 담백함, 문장을 닮은 안정감

### Medium · 500

> **산을 닮은 담백함, 문장을 닮은 안정감**

```text
가나다라마바사아자차카타파하
ABCDEFGHIJKLMNOPQRSTUVWXYZ
abcdefghijklmnopqrstuvwxyz
0123456789

오늘의 문장을 또렷하게 보여주는 일상형 한글 폰트
```

## 웹에서 사용하기

```css
@font-face {
  font-family: "Jayeon Sans";
  src: url("./JayeonSans-Regular.ttf") format("truetype");
  font-style: normal;
  font-weight: 400;
  font-display: swap;
}

@font-face {
  font-family: "Jayeon Sans";
  src: url("./JayeonSans-Medium.ttf") format("truetype");
  font-style: normal;
  font-weight: 500;
  font-display: swap;
}

body {
  font-family: "Jayeon Sans", sans-serif;
}
```
